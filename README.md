# Learning Rust

## Using Cargo to manage Rust projects

### Create a new project

```bash
cargo new my-project
```

### Build the project

```bash
cargo build
```

### Build and Run the project

```bash
cargo run
```

### Test the project

```bash
cargo test
```

### Build for Release

```bash
cargo build --release
```

### Publish the project

```bash
cargo publish
```
